Parch

Decentralized messaging, voice & video


A suite of applications for real-time communication via a relay service. User-hosted, privacy-focused, and completely free.

Three Applications

Parch Client

The main chat application. Connect to multiple hosts, join spaces, and communicate through channels.

Parch Host

Run your own server with SQLite storage. Generate a Host Key to share with users who want to connect.

Parch On-Demand

Free voice and video conferencing in your browser. No downloads or accounts required.

Start a call →

Getting Started

Join as a Client

If you have a Host Key from a host provider, download the Client application below. You'll be invited to "spaces" where you can chat in channels created by the space owner.

Run Your Own Host

Download the Host application and run it to register with the relay service. You'll receive a Host Key to share with anyone you want to connect.
